2. **CalendarUtils.kt**
   - 修复：移除未使用的context参数
   - 影响：代码更简洁，无警告

### 性能优化措施
1. **列表优化**
   - 使用 `key = { it.id }` 优化列表项重用
   - 使用 `contentType = { "schedule" }` 标识列表项类型
   - 使用 `animateItemPlacement()` 优化动画性能

2. **网络优化**
   - 连接超时：30秒
   - 读取超时：30秒
   - 写入超时：30秒

3. **内存优化**
   - 使用StateFlow替代LiveData
   - 使用WhileSubscribed(5000)自动停止上游Flow
   - 使用Application Context避免内存泄漏
